# Autumn colorscheme

VIM colorscheme with code based on that of Tomorrow-Night-Eighties

![screenshot](screenshot.png)

## Installation

### Using Vundle

```
Plugin 'https://gitlab.fi.muni.cz/xkana1/vim-wollf.git'
```

`:colorscheme autumn`
